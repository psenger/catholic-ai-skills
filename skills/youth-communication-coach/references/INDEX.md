# Saints & Martyrs Reference Index

This index lets the `youth-communication-coach` skill find the right saint for a coaching situation **without loading every entry into context**. Find the row that matches the topic, age group, virtue, or Beatitude the conversation calls for. Read only the single saint file linked from that row.

Each saint has its own file at `references/saint-<slug>.md` — typically 20–30 lines, containing: feast day, beatitude(s) and virtue(s), age at death, biography, documented and attributed quotes, "The Point" (the argument the saint makes), "In Conversation" (a natural opener), and recommended age group.

**Quote attribution key:** *Documented* = verifiable in primary sources. *Attributed* = traditional attribution; exact wording may vary.

---

## How to use this index

1. Identify the coaching need: a topic (death, dating, failure), an age group (child, teen, young adult), a virtue, or a Beatitude.
2. Find the matching section below and pick the saint whose story fits.
3. Open the linked file — that single file contains everything you need to use the saint in conversation.
4. Use "In Conversation" as your opener; draw on the quotes and "The Point" to develop the argument.

If a saint is listed in a cross-reference here but does NOT have a file (no link), that figure is referenced in passing but not yet fully documented in this library. For richer coverage of those saints, the sibling `catholic-advisor` skill's reference library may have an entry — check there.
